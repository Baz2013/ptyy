# 莆田系医院查询

[![GoDoc](https://godoc.org/github.com/chai2010/ptyy?status.svg)](https://godoc.org/github.com/chai2010/ptyy)

Install
=======

1. `go get github.com/chai2010/ptyy`
2. `go run hello.go`

Example
=======

```Go
package main

import (
	"fmt"

	"github.com/chai2010/ptyy"
)

func main() {
	fmt.Println(ptyy.All)
}
```
